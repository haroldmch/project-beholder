# Project-Beholder

Project-Beholder, is a project that aims to create a multi-tool for playing D&D. 

Objectives:
- Character creator 

## Getting Started

Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
